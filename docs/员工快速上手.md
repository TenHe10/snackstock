# SnackStock 员工快速上手（1页）

## 1. 每天开店先做这 3 件事

1. 启动系统：`uv run python main.py`
2. 看启动预警弹窗（缺货/临期）
3. 进入 `库存与报表` 页，确认库存表是否正常

---

## 2. 入库（来货时）

进入 `入库` 页：

1. 先看商品是否已存在  
如果是新商品，先在“商品档案”填写并点 `新增/更新商品`
2. 在“入库扫码”扫码/输入条码
3. 填数量、批次、过期日期
4. 点 `确认入库`

要点：

- 批次和过期日期建议每次都填，后续临期提醒更准确

---

## 3. 出库（收银时）

进入 `出库` 页：

1. 扫码（回车后自动加入购物车）
2. 重复商品会自动累加数量
3. 顾客结账时点 `结算`

要点：

- 如果提示库存不足，先核对库存或改数量再结算
- 如果提示商品不存在，去 `入库` 页先建档

---

## 4. 看预警与日报

进入 `库存与报表` 页：

- `预警框`：看缺货商品和临期商品
- `日报`：看营业额 / 进货额 / 毛利润
- 需要留档时点 `导出日报 CSV`

---

## 5. 常见问题快速处理

### 扫码没反应

1. 确认光标在条码输入框
2. 确认扫码枪是键盘模式（HID）
3. 试一次手输条码+回车

### 导出失败

1. 换一个有权限的目录
2. 关闭已打开的同名 CSV 再导出

---

## 6. 交接班口径

- 交班前导出当天日报 CSV
- 每日备份一次 `database/inventory.db`
